## Instruções de Uso

Este template ajuda a transformar análise comparativa em hipóteses acionáveis de diferenciação.

**Como usar:**
1. Revise síntese comparativa (Etapa 5) e contexto de negócio (Etapa 1)
2. Para cada gap/oportunidade identificada, gere hipótese
3. Use template estruturado para cada hipótese
4. Avalie viabilidade, riscos e alinhamento
5. Priorize top 2-3 hipóteses
6. Defina próximos passos

**Meta:** 5-10 hipóteses explorando diferentes dimensões de diferenciação

---

## TEMPLATE DE HIPÓTESE

Use este formato para cada hipótese:

**"Nosso produto resolve [PROBLEMA] para [ICP ESPECÍFICO] através de [ABORDAGEM DIFERENCIADA], diferente dos competidores que [COMO COMPETIDORES FAZEM]."**

---

## DIMENSÕES DE DIFERENCIAÇÃO

Explore pelo menos 3 das 4 dimensões:

### A. Problema/ICP Diferente
- Mesmo problema para ICP desatendido
- Subproblema específico negligenciado
- Nicho vertical, geografia ou tamanho

### B. Solução Diferente
- Abordagem técnica distinta
- UX radicalmente superior ou simplificada
- Fazer menos features, mas muito melhor

### C. Modelo de Negócio Diferente
- Pricing mais acessível ou transparente
- Modelo de cobrança alternativo (freemium, usage, etc.)
- Go-to-market ou distribuição diferenciada

### D. Posicionamento Diferente
- Linguagem e branding distintos
- Benefício negligenciado pelos competidores
- Contrarian positioning (ir contra consenso)
