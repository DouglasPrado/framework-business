## Templates a serem criados durante o processo

### 1. roadmap-secundarias.md
**Criado em:** Etapa 1 - Planejamento e Priorizacao
**Proposito:** Roadmap priorizado de funcionalidades secundarias
**Conteudo:**
- Lista de funcionalidades secundarias
- Score de valor e esforco de cada uma
- Ordem de implementacao
- Timeline estimada
- Dependencias tecnicas

### 2. criterios-aceite-secundarias.md
**Criado em:** Etapa 1 - Planejamento e Priorizacao
**Proposito:** Criterios de aceite detalhados para cada funcionalidade secundaria
**Conteudo:**
- Para cada funcionalidade secundaria:
  - Criterios tecnicos
  - Criterios funcionais
  - Criterios de qualidade
  - Exemplos de uso

### 3. eventos-analytics.md
**Criado em:** Etapa 7 - Integracao de Analytics e Monitoramento
**Proposito:** Documentacao de eventos rastreados no analytics
**Conteudo:**
- Lista de eventos customizados
- Parametros de cada evento
- Quando cada evento e disparado
- Exemplos de uso

### 4. email-templates/ (pasta, se usar notificacoes por email)
**Criado em:** Etapa 4 - Sistema de Notificacoes
**Proposito:** Templates HTML de emails transacionais
**Conteudo:**
- Templates de email para cada tipo de notificacao
- Usar React Email ou HTML simples
- Incluir versoes em texto plano (fallback)
