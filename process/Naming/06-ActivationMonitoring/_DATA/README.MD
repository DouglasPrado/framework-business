# _DATA - Activation and Monitoring (Naming)

Armazene aqui todos os artefatos relacionados à ativação do nome e ao monitoramento contínuo.

## Estrutura sugerida

```
_DATA/
  06-ActivationMonitoring/
    plano-ativacao-nome.MD
    checklist-atualizacao-materiais.MD
    treinamento-nome.MD
    comunicados/
    evidencias/
      atualizacoes-materiais/
      comunicados-externos/
      treinamentos/
    monitoramento-marca-nome.MD
    log-ajustes-nome.MD
```

## Templates recomendados

- Planos e checklists de Branding (ExperienceJourney, RolloutGovernance) adaptados para naming.
- Ferramentas de dashboards (Sheets, BI) aprovadas em `tools/`.

## Boas práticas

- Versionar plano e checklist; registrar data, responsável e status.
- Salvar evidências (prints, links, vídeos) na pasta `evidencias/`.
- Atualizar `log-ajustes-nome.MD` após cada ritual de monitoramento.
