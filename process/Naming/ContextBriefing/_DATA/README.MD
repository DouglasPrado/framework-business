# _DATA - Context Briefing (Naming)

Organize aqui os artefatos gerados durante a etapa de Context Briefing do processo de Naming.

## Estrutura sugerida

```
_DATA/
  ContextBriefing/
    briefing-naming.MD
    stakeholders-list.MD
    entrevistas-iniciais.MD
    criterios-avaliacao.MD
    riscos-dependencias.MD
    cronograma-naming.MD
```

## Templates recomendados

- `strategies/Naming/_DATA/templates/briefing-naming.MD`
- `process/_SHARED/templates/script-entrevista.MD`
- Outputs de branding (persona, posicionamento, identidade verbal) para referência

## Boas práticas

- Nomeie arquivos com padrão `YYYY-MM-DD_nome.MD`.
- Registre fontes de dados (links, gravações, planilhas) com responsável.
- Atualize o briefing imediatamente após cada reunião para evitar perda de contexto.
