# _DATA - Generation and Consolidation (Naming)

Organize nesta pasta os artefatos gerados durante a etapa de geração e consolidação de nomes.

## Estrutura sugerida

```
_DATA/
  02-GenerationConsolidation/
    longlist-naming.MD
    log-sessoes-criativas.MD
    descartes-naming.MD
    priorizacao-longlist.MD
    ideias-rapidas.MD
    combinacoes-automatizadas.MD
```

## Templates recomendados

- `strategies/Naming/_DATA/templates/longlist-naming.MD`
- `strategies/Naming/_DATA/templates/matriz-inspiracao.MD` (referência de campos semânticos)
- Outputs de `ResearchInputs` (`banco-palavras-chave.MD`, `matriz-inspiracao.MD`, `alertas-linguisticos.MD`)

## Boas práticas

- Nomeie arquivos com padrão `YYYY-MM-DD_nome.MD` ou utilize versionamento (`longlist-naming_v1.MD`).
- Registre técnicas e aprendizados em `log-sessoes-criativas.MD`.
- Anote motivo de descarte imediatamente para evitar que opções retornem sem contexto.
