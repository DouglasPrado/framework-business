# _DATA - Research and Inputs (Naming)

Organize nesta pasta todos os artefatos produzidos durante a etapa de pesquisa e insumos.

## Estrutura sugerida

```
_DATA/
  ResearchInputs/
    matriz-inspiracao.MD
    banco-palavras-chave.MD
    benchmarks-naming.MD
    alertas-linguisticos.MD
    relatorio-pesquisa-naming.MD
    fontes-referencia.MD
```

## Templates recomendados

- `strategies/Naming/_DATA/templates/matriz-inspiracao.MD`
- `strategies/Naming/_DATA/templates/longlist-naming.MD` (usar como referência para categorias de nomes)
- `process/_SHARED/templates/script-entrevista.MD` (se realizar entrevistas adicionais com clientes/usuários)

## Boas práticas

- Nomeie arquivos com padrão `YYYY-MM-DD_nome.MD`.
- Registre a fonte de cada referência (link, print, nota) em `fontes-referencia.MD`.
- Classifique referências como positiva/negativa/neutra para acelerar filtros posteriores.
