# _DATA - Experience and Journey

Diretório para armazenar os artefatos produzidos durante o processo ExperienceJourney.

## Organização sugerida

```
_DATA/
  ExperienceJourney/
    jornada-mapeada.MD
    playbook-canais/
      canal-site.MD
      canal-social.MD
      canal-vendas.MD
      canal-suporte.MD
    checklist-experiencia.MD
    indicadores-experiencia.MD
    feedback-log.MD
    plano-governanca-experiencia.MD
    agenda-ritual-governanca.MD
```

## Templates recomendados

- `strategies/Branding/_DATA/templates/guia-consistencia-canais.MD`
- `strategies/Branding/_DATA/templates/plataforma-de-marca.MD`
- `process/_SHARED/templates/checklist-qualidade.MD`
- Planilhas de métricas ou dashboards aprovados (ver diretório `tools/`)

## Boas práticas

- Nomeie arquivos com padrão `YYYY-MM-DD_nome-arquivo.MD`.
- Registre links para materiais externos (dashboards, gravações) com responsável e data.
- Utilize subpastas por canal para facilitar manutenção e auditoria.
