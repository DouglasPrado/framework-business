# _DATA - Rollout and Governance

Diretório para armazenar os artefatos gerados durante o processo RolloutGovernance.

## Organização sugerida

```
_DATA/
  RolloutGovernance/
    plano-rollout.MD
    kit-lancamento-interno/
      apresentacao.MD
      faq.MD
      comunicado-email.MD
      roteiro-townhall.MD
    calendario-rollout-externo.MD
    dashboard-marca.MD (ou links)
    plano-governanca-marca.MD
    agenda-governanca.MD
    feedback-log.MD
    relatórios/
      monitoramento-marca-YYYY-MM.MD
```

## Templates recomendados

- `strategies/Branding/_DATA/templates/guia-consistencia-canais.MD`
- `process/_SHARED/templates/checklist-qualidade.MD`
- Modelos de comunicados e apresentações aprovados pelo time de marketing
- Ferramentas listadas em `tools/` para dashboards e monitoramento

## Boas práticas

- Versione documentos com padrão `YYYY-MM-DD_nome-arquivo.MD`.
- Mantenha links para dashboards, ferramentas e repositórios externos com permissões corretas.
- Atualize registros de governança imediatamente após reuniões ou marcos.
