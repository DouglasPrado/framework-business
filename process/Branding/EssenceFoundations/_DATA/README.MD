# _DATA - Essence and Foundations

Este diretório concentra os artefatos gerados durante o processo de EssenceFoundations no Branding.

## Organização sugerida

```
_DATA/
  EssenceFoundations/
    plataforma-de-marca.MD
    workshop-notes.MD
    narrativa-essencia.MD
    valores-exemplos.MD
    agenda-posicionamento.MD
```

## Templates recomendados

- `strategies/Branding/_DATA/templates/plataforma-de-marca.MD`
- `strategies/Branding/_DATA/templates/guia-consistencia-canais.MD` (para manter alinhamento com experiências futuras)
- `process/_SHARED/templates/script-entrevista.MD` (adaptar para sessões internas, se necessário)

## Boas práticas

- Nomeie arquivos com padrão `YYYY-MM-DD_nome-arquivo.MD`.
- Salve fotos ou gravações de workshops em repositórios aprovados e referencie os links.
- Atualize a pasta imediatamente após cada sessão para evitar perda de contexto.
