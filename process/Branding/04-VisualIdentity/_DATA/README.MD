## Templates recomendados

## Boas práticas

- Nomeie arquivos seguindo `YYYY-MM-DD_nome-arquivo.ext`.
- Mantenha links para arquivos pesados (Figma, Drive) com permissão de acesso.
- Documente alterações em `feedback-log.MD` para versionamento.
