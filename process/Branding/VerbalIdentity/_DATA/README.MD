## Boas práticas

- Use nomenclatura `YYYY-MM-DD_nome-arquivo.MD`.
- Versões intermediárias devem ser arquivadas em subpasta `archive/`.
- Registre links de materiais externos (gravações, planilhas) com responsável e data.
