# _DATA - Imersão e Contexto

Este diretório armazena os artefatos gerados durante a execução do processo de Imersão e Contexto do Branding.

## Organização sugerida

```
_DATA/
  ContextImmersion/
    entrevistas-internas.csv
    entrevistas-externas.csv
    benchmarking.MD
    swot.MD
    diagnostico-resumo.MD
```

## Templates recomendados

- `strategies/Branding/_DATA/templates/diagnostico-marca.MD`
- `strategies/Branding/_DATA/templates/guia-consistencia-canais.MD` (para mapear referências de canais relevantes)
- `process/_SHARED/templates/script-entrevista.MD`
- `process/_SHARED/templates/persona.MD`

Ao iniciar o processo, copie o template de diagnóstico para `_DATA/ContextImmersion/diagnostico-marca.MD` e utilize as planilhas/notas sugeridas para registrar entrevistas e benchmarking.
