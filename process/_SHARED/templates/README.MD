## Templates Compartilhados Disponíveis

Use estes arquivos como base ao criar materiais operacionais. Duplique para a pasta `_DATA/` do processo correspondente antes de personalizar.

- `declaracao-hipotese.MD` – estrutura para formular hipóteses de problema/solução.
- `script-entrevista.MD` – roteiro padrão para entrevistas de validação e discovery.
- `persona.MD` – ficha detalhada de persona para segmentação e copy.
- `email-followup.MD` – variações de follow-up (outreach, pós-call, pós-entrega, reengajamento).
- `estrutura-landing.MD` – blueprint textual de landing page com seções obrigatórias.
- `checklist-qualidade.MD` – checklist unificado de QA para páginas, fluxos e entregáveis.
- `retrospectiva.MD` – registro oficial da retrospectiva quinzenal e ações.

> Sempre que um processo utilizar um destes templates, referencie o caminho relativo `../_SHARED/templates/<arquivo>.MD` e mantenha a versão preenchida dentro de `_DATA/` para evidências.
