## Documentos Criados

### 1. Plataforma de Marca ([01-plataforma-de-marca.MD](01-plataforma-de-marca.MD))
**Documento central** com todos os elementos da essência:

#### Propósito
**"Devolver clareza financeira para quem produz com as próprias mãos"**

#### Visão (2028)
**"Todo MEI brasileiro sabe exatamente quanto pode retirar do próprio negócio — e retira com tranquilidade"**

#### Missão
**"Iluminamos o caminho financeiro de MEIs através de tecnologia inteligente + cuidado humano"**

#### 5 Valores
1. **Clareza Radical** - Mostramos a verdade, sempre
2. **Cuidado Constante** - Acompanhamos toda semana, protegemos o caixa
3. **Proximidade Humana** - Falamos como gente, não como software
4. **Simplicidade Prática** - Fazemos o difícil parecer fácil
5. **Empoderamento Financeiro** - Ensinamos, não criamos dependência

#### Personalidade
**Arquétipo**: Sábia Cuidadora (Sage + Caregiver)
**Tom**: Amiga experiente, inteligente mas acessível, cuidadosa sem ser paternalista

#### Diferenciadores
- Híbrido perfeito (IA + Humano)
- Foco em clareza, não controle
- Personagem humanizada (Lúmina existe)
- Comunicação em português humano
- Começamos pelo mais simples

## Destaques da Essência

### O que torna Lúmina única

**1. Humanização perfeita**
- Lúmina não é apenas um "serviço"
- É uma personagem feminina que cuida e ilumina
- IA + Ana Luiza = híbrido tech + humano

**2. Clareza como valor emocional**
- Não vendemos "controle financeiro"
- Vendemos "alívio" de saber quanto pode tirar
- Resolvemos 1 problema profundamente

**3. Tom validado**
- Amiga, não consultora
- Próxima, não corporativa
- Simples, não técnica

## Aplicação Prática

### Como usar a essência em diferentes áreas

**Marketing e Growth**
- Toda peça deve refletir **clareza + cuidado**
- Tom sempre **próximo**, nunca corporativo
- Benefício **emocional** (alívio) > funcional (planilha)

**Produto**
- UX **simples** como WhatsApp
- Alertas **proativos**, nunca reativos
- Resultado **primeiro**, explicação depois

**Atendimento (Ana Luiza)**
- Sempre **empática**, nunca apressada
- Explicar **quantas vezes** for necessário
- **Celebrar** progressos do cliente

**Vendas / Outreach**
- Pitch: "Mostramos quanto você pode tirar sem quebrar"
- Foco na **dor** antes do produto
- Usar **linguagem validada** com clientes

## Comportamentos por Valor

### Clareza Radical
✅ Usar números reais, não estimativas
✅ Explicar o "porquê" de cada alerta
✅ Avisar ANTES do problema
❌ Respostas genéricas
❌ Esconder más notícias
❌ Jargões técnicos

### Cuidado Constante
✅ Alertas semanais proativos
✅ Avisar ANTES do dinheiro acabar
✅ Responder rápido (< 2h)
❌ Sumir após onboarding
❌ Alertas genéricos
❌ Deixar cliente sem resposta

### Proximidade Humana
✅ Usar "você", "vamos", "juntos"
✅ Celebrar conquistas
✅ Reconhecer dificuldades
❌ Linguagem corporativa fria
❌ Tom de professor superior
❌ Ignorar contexto emocional

### Simplicidade Prática
✅ Resultado final primeiro
✅ Linguagem cotidiana
✅ Funcionar no WhatsApp
❌ Exigir planilhas complexas
❌ Dar 10 opções quando 1 resolve
❌ Ensinar contabilidade desnecessária

### Empoderamento Financeiro
✅ Explicar o "porquê"
✅ Mostrar evolução
✅ Ensinar conceitos simples
❌ Esconder lógica
❌ Infantilizar
❌ Criar processos obscuros

## Vocabulário Essencial

### SEMPRE usar
- Lucro mensal claro
- Retirada ideal
- Saldo à vista
- Separar o que é meu do que é da empresa
- Dinheiro que não some
- Iluminar / mostrar / enxergar
- Sem medo / sem culpa / sem preocupação

### NUNCA usar
- Jargões contábeis (balancete, DRE, conciliação)
- Termos negativos (dívida, bloqueio, corte)
- Burocracia (imposto direto, alvará, fiscalização)
- Linguagem bancária fria
- Promessas exageradas

## Critérios de Validação

Antes de aprovar, verificar:

**Propósito**
- [ ] Responde "por que existimos?"
- [ ] Conecta com dor validada dos clientes
- [ ] Inspira a equipe

**Visão**
- [ ] Descreve futuro desejado (3-5 anos)
- [ ] É aspiracional mas alcançável
- [ ] Todos entendem o que mudar no mundo

**Missão**
- [ ] Explica o que fazemos hoje
- [ ] Deixa claro para quem e como
- [ ] Diferencia de concorrentes

**Valores**
- [ ] Cada valor tem ≥ 3 comportamentos observáveis
- [ ] Comportamentos são acionáveis ("fazer/evitar")
- [ ] Equipe consegue aplicar no dia a dia

**Personalidade**
- [ ] Arquétipo faz sentido para o público
- [ ] Tom funciona em todos os canais
- [ ] Exemplos práticos são claros
