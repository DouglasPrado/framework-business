`[Confirmação] Seu pagamento para Automarticles foi recebido`

1. **Saudação personalizada**
   ```
   Olá, <Nome do Cliente>!
   ```

2. **Confirmação do pagamento**
   ```
   Confirmamos o recebimento do seu pagamento referente a Automarticles, realizado em <Data> no valor de R$ 99,00.
   ```

3. **Resumo do pedido**
   - Oferta/Plano: Automarticles
   - Valor pago: R$ 99,00
   - Método de pagamento: <Método>
   - Número do pedido / ID da transação: <ID da transação>

4. **Próximos passos**
   ```
   Nas próximas 24 horas você receberá um e-mail de confirmação com os detalhes da sua compra. Caso precise falar conosco, utilize o e-mail contato@automarticles.com.
   ```

5. **Acessos ou materiais imediatos**

   - Página de obrigado / FAQ: [https://example.com/automarticles/obrigado](https://example.com/automarticles/obrigado)

6. **Suporte**
   ```
   Qualquer dúvida, responda este e-mail ou entre em contato pelo e-mail contato@automarticles.com.
   ```

7. **Assinatura**
   ```
   Obrigado,
   Equipe Automarticles
   Suporte
   ```
