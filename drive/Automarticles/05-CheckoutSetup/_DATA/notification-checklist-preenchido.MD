# Checklist de Notificações e Integrações

Preencha este checklist após realizar testes de pagamento. Marque cada item com a evidência correspondente (print, log, e-mail).

## E-mails Automáticos do Gateway

- [x] Cliente recebe e-mail de confirmação com dados corretos (anexar print)
- [x] Time interno recebe notificação de nova venda
- [x] Link de recibo/nota fiscal presente (quando aplicável)
- [x] Conteúdo revisado (linguagem, instruções, dados de contato)

## Webhooks / Automação

- [x] Webhook para CRM/Slack disparou com payload correto
- [x] Automação cria registro em planilha ou ferramenta aprovada
- [x] Logs verificados sem erros (ID do evento registrado)
- [x] Falhas tratadas com plano de contingência

## Planilha de Controle

- [x] Entrada manual ou automática registrada
- [x] Colunas obrigatórias preenchidas (cliente, valor, status, recibo enviado)
- [x] Responsável notificado sobre nova venda

## Alertas Manuais

- [x] Mensagem automática ou manual enviada para time de entrega
- [x] Cliente recebe instruções adicionais (quando necessário)
- [x] Próximos passos definidos e comunicados

## Observações

- Pendências encontradas: Não informado
- Ações corretivas: Não informado
- Próxima revisão agendada para: Não informado
